# cruise-product-info · Claude Code 插件

获取特定邮轮产品的详细信息，了解产品的特点和服务，帮助决策。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-product-info-claude-plugin
/plugin install cruise-product-info@cruise-product-info-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-product-info**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
